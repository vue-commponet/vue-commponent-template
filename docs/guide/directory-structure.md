# 目录结构

```
.
├── docs // 文档说明，参考vuepress
├── src // 组件开发目录
│    ├── index.js // 组件注册文件，参考Vue组件开发文档 
│    └── ***.vue // 组件逻辑
│
├── examples // 组件演示目录
│    ├── main.js // 入口文件
│
├── .babelrc // babel-loader配置文件
├── postcss.config.js // postcss-loader配置文件
├── .gitignore // git忽略文件
├── .component.config.js // 组件信息配置文件
├── webpack.config.js // webpack配置文件
├── README.md
├── package-lock.json
└── package.json

```
